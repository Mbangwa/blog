Variables

Lets go aheaaddddd.......

Tunajua kwamba kila computer ina Memory na CPU. Ili kuchakata data lazima computer iwe na sehemu ya kutunza data ambazo znatakiwa kuchakatwa na zile ambazo tayari zmeshachakatwa. Apo ndo inakuja umuhimu wa kua na memory kubwa ndan ya computer.

Memory yenyewe imegawanywa katika segments ndogo ndogo zenye ukubwa sawa zote! Na kila segment imepewa namba maalumu inaitwa Memory address. mfano @x67f343g2 Chukulia mfano wa Post office, kuna visanduku vingii vyenye ukubwa sawa na kila kisanduku kina namba yake ambyo ni tofaut na vsanduku vngne... Ata memory ndvo ilivo.

But ikaonekana ni ngumu kukalili majina ya kila memory segment kama ilivyo apo. So wakaamua kuunda kitu knachoitwa Variable.

A variable is nothing but A name of memory segment!.
So if u wanna store data in computer memory, u should first tell the computer what kind of data u want to store and in which memory segment u want ur data to be stored.

By doing so, We say that "You declare a variable."

Example on variable declaration
int eyes;
double weight;
String name;
boolean isTall;
char gender;

Maneno int, double, String, boolean na char ndo yanaiambia computer ni aina gani ya data unataka kutunza.
int for integers (Namba nzima) mfno 30
double for decimal numbers (Namba zenye desimali) mfno 9.99
String for Strings (Maneno) mfno "mama"
boolean for boolean values (kweli au sikwel) mfno true or false
char for characters (herufi na alama) mfno 'Z'

Maneno eyes, weight, name, isTall na gender Ndo varables, kwa maana nyngne ndo majna ya memory segment, badala ya kutumia zile memory address sasa unaweza kuipa memory segment jina lolote kama tulivofanya apo! its very sweet and clear.
But ohhh...wait,,, jina lolote? NOOO not always,, kuna sheria za kuzipa majina memory segments... tutaona apo mbele kwenye 'rules of naming variables'

Unapoanza kuassign data kweny variable tunaita Initialization
Example on variable initialization
int eyes = 2;
double weight = 4.6;
String name = "juma";
boolean isTall = true;
char gender = 'M';
As u see, tunatumia '=' (assigment operator kufanya initialization).
Hapo unakua umetunza data izo kwenye izo variable.

Rules of naming a varible..

  1. Jina la variable haitakiwi liwe namba tu. Mfno int 123 = 678; (ni kosa)
  2. Jina la variable halitakiwi kuanza na namba. Mfno int 1age = 103; (ni kosa)
  3. Jina la variable lisiwe sawa na maneno ambyo yametunzwa na Java language, mfno wa ayo maneno ni int, String, final, extends, implements, boolean, etc.
  4. Usitumie special charaters kwe varables mfno int +number = 78; (ni kosa)
  5. Tumia neno final kuifanya variable iwe constant (isiweze badilishwa tena). Mfno final double PI = 3.14;
