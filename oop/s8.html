Inheritance

Inheritance is sweet, it minimizes the time for typing the same codes over and over again.
The rule is, never repeat yourself.

Inheritance ni kurithi.
Object moja inaweza kurithi tabia za object nyingine.

Tumia neno final kuzuia class isirithiwe tabia zake.
Tumia neno extends kurithi tabia za class flani.
Class ambayo imerithiwa tabia zake inaitwa super class au parent class
Class ambyo imerithi tabia za class flani inaitwa subclass or child class.
Kumbka, constructors hua haziwezi rithiwa, so kama una class ambyo ina constructor na unataka kurithi tabia zake bhac lazima uiite io constructor kweny child or sublclass kwa kutumia neno super, otherwise u gonna have an error.

Mfno:
tunaeza tengeneza class Plant ikarithi class Animal, so class plants itakua na tabia zote za animal, na ikitaka kujiongezea pia itajiongezea tabia zingne.
public class Plant extends Animal {

public Plant(int l, double w){
super(l,w); //tunatumia super keyword to call a superclass constructor.

}
}

In java it is not possible to assign one reference type to another except for superclass and subclass.
Here we mean, lazima class mbili ziwe na uhusiano wa kurithian ndo utaweza assign object moja ya super na object ingne ya sub-class.

Method overriding.

Method overriding is simpo,, just kubadli statement za kweny method body za methods zilizo kweny superclass ukiwa ndani ya subclass without changing the signature of that method.
