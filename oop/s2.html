Input and Output (I/O).

Sote tunajua kwamba program ya computer yoyote hua inahitaj kuingiza taarifa flani ili izichakate and then ikupe majibu, and probably all computers were made for that purpose, "Processing data, storing them and outputing them when needed".

In java we have mechanisms for collecting data from the user and mechanisms for outputing data to the user. We call 'em Input and Output (I/O) systems.

Kuna class nyingi ambazo zina methods(functions) za kuchukua data from the user na pia kudisplay data to the user. Ila class ambayo ni common na simple kuitumia ni Scanner class.

Let's see how it works...

The Scanner class

The Scanner class is the Input systemspecial class which comprises so many methods which are non-static.
Unakumbka nlisemaje? Static methods don need the creation of object to use them.
Lakin apa Scanner class ni non-static class which means we have to screate its object in order to use it's methods.

Pia Scanner class is not imported automatically, so we need to import it before duin anything.
To import it we use this way:
import java.util.Scanner;
kwenye io statement, java.util ni package ambayo imebeba Scanner class.

How to use the Scanner class...

Ili kuitumia Scanner class unahtajika kwanza kutengeneza object ya io class. Tunatumiaga new keyword kuunda object from a class. So lets see an example:
Scanner scan = new Scanner(System.in);
Hio statement ukiichek utaona imeanza kutaja kwanza jina la class (Scanner), then ikafatiwa na neno scan, Hili neno ndo object yetu, unaweza eka neno lolote upendalo, sio lazima neno 'scan' kama nlivochagua mim... Then kuna '=' after io sign, kuna neno 'new', ili neno ni lazima liwepo na linaandikwa kama lilivo, after ilo neno kuna jina la class tena with parameters.

Apo taari tumetengeneza object iitwayo scan. Tunaweza kuitumia kuchukua data from the user of the program. But howwww? don panic, watch....
Tunatumia "dot" syntax kupata methods zote ambzo zpo ndani ya io object. look..
scan.nextInt();
-> Tunaanza na jina la object, then "dot", Then jina la method kulingana na data unayotaka ichukuliwe. "nextInt()" method inachukua integer data types only, so kama unataka kuchukua "String data" use methods like "next()", or "nextLine()".
scan.next(); or scan.nextLine();
Owhhh...So easy. so far so good.
