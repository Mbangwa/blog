Simplest java program..

public class Animal{

public static void main(String[] args){
System.out.println("Animals can eat.");
}
}

Tumesema hivii, kila program ya java lazima iwe ndani ya class flani, ndomana Tunasema java ni pure OOP language. Bila class huezi andka program ya java.

Now tuichambue kama karanga io program rahisi kulko zote duniani.

public class Animal {} -> Neno public ni access modifier, means linaipa class uwezo wa kuonekana na class zingine ambazo utatengeneza. Neno class baada ya public ndo linaiambia compiler kua iki kitu nnachotengeneza ni class. Na mbele ya neno class itafatiwa na jina la io class unayotaka kutengeneza. Mfano apo nmetumia neno Animal kama class name.

public static void main(String[] args){} -> Pia apa matumizi ya neno public ni kama ya kwe class. Neno static linamaanisha "hii method haitohitaji wala kutegemea utengenezaji wa object ya hii class". Tunajua kwamba kila ukitengeneza class nje ya main program lazima utengeneze object ya ile class kwanza ndo ueze itumia. But ukidefine method or class as static haitohtaj utengeneze object.
This is so beautiful......
Neno main(String [] args) linatamblisha kua ii ni method kuu (main method). Ili computer itambue ipi ni program na ipi ni class ua inaangalia uepo wa main method ikiwa na parameter zake ndani i.e String [] args. Hizi parameters znamanisha kua ii method inaweza chukua arrays of strings kutoka kwe terminal console. Kama ujaelewa apa tulza munkari... mbele utaelewa tuuuu.

System.out.println("Animals can eat."); System.out ni package kama packages ambazo hua unatengneza ukiwa unataka kutengeneza project flani ndani ya editor kam eclipse. Na ndani ya io package kuna class ambayo imebeba methods nyingi tu.. but method hii println() ni method ambyo inauwezo wa kutoa output ya chochot utachoandika ndani yake, either a varible, string or object! So apa maneno "Animals can eat" yatatoka kama output.
Naa, mwsho unaona kuna semicolon, ii ndo inakamilisha statement. Bila io compiler haitoelewa mwsho wa statement yako ni upi, kwo automatically itakupa error
