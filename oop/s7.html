...objects

In the previous section we saw the creation of a class Animal. Now lets look how can we create an object from that class..
The class Animal looked like this:
public class Animal {

private int legs;
private double weight;

//constructor
public Animal(int l, double w){
legs = l;
weight = w;
}

//Accessors
public int getLegs(){
return legs;
}

public double getWeight(){
return weight;
}

//Mutators
public void setLegs(int legs){
this.legs = legs;
}
public void setWeight(double weight){
this.weight = weight;
}
}


So, io class utaona ina constructor inayohtaj parameter mbili, 'l' na 'w'..
Ua tunatengeneza objects ndani ya main class (program)... kwaio tutengeneze main class.

public class Main{

public static void main(String[] args){

}
}

Ivo ndo tunatengeneza main class, lazima iwe na main method public static void main(String[] args) {}.
Ndani ya main method ndo tunaandika code za kuunda object, na kama unakumbka nlsema ili kuunda object ua tunatumia neno "new".
Now lets see how....
public class Main{

public static void main(String[] args){

Animal dog = new Animal(4, 17.5);

}
}
 Umeona?
Upande wa kushoto kabla ya '=' umebeba object reference of type 'Animal' i.e Animal dog, kwaio dog ndo object reference yetu.
Upande wa kulia wa '=' ndo tumetengeneza object na kupitisha values 4, na 17.5, izo value ndo ztaenda kwenye 'legs' na 'weight'.

So,ili kuanza kuchange or kuaccess values tulizoset ndo tunatumia zile getters na setters. Getters kwa ajili ya kuaccess na setters kwa ajili ya kuchange.

public class Main{

public static void main(String[] args){

Animal dog = new Animal(4, 17.5);

dog.getLegs(); // will return 4
dog.getLegs(); // will return 17.5

}
}
 So, ili kuchange izo values tunatumia setters instead of getters.

public class Main{

public static void main(String[] args){

Animal dog = new Animal(4, 17.5);

dog.setLegs(2); // will set 2 into legs
dog.setLegs(30); // will set 30 into weight

dog.getLegs(); // will return 2
dog.getLegs(); // will return 30

}
}
 Now the values of 'legs' and 'weight' are 2 and 30 respectively, not 4 and 17.5 as before!

Note: Many ppl get confused when they run a program like that and witness nothing in the output console....
This is bcoz getters just returns the value to the caller, and won't display anything tho the console.
If you wanna see the return value, create a variable to store the returned value and use the output method to display them.

}
