1 (a).Write a java class named Stats that has one data member of type array of int and two methods one calculate the maximum number and another calculate the average number.
(b).Write the java program named StatsMain that make use of Stats class by allowing user to enter numbers and then use the above class to compute maximum and average number

Solution:

Stats class:
/**
* @author Mbangwa (inno).
*
*/
public class Stats {
// one data member of type array of int

int[] array;

public Stats(int size) {
// array initialization
array = new int[size];
}

// method for average

public double average() {

double total = 0;
double average = 0;

for(double value: array) {
total += value;
}

average = total/array.length;
return average;
}

// maximum finder method

public double maximum() {

// assuming the maximun number is the first number in our array

double maximum = array[0];
for (double value: array) {
if (value > maximum) {
maximum = value;
}
}
return maximum;
}
}

StatsMain class:

import java.util.Scanner;
/**
* @author Mbangwa (inno).
*
*/

public class StatsMain {
public static void main(String[] args) {
int size;
int count = 0;
Scanner scan = new Scanner(System.in);
System.out.print("Enter size of an array: ");
size = scan.nextInt();

// lets create an object of Stats class.
Stats stat = new Stats(size);

//prompt the user to enter values
System.out.println("Enter values");
while(true) {
System.out.print(">> ");
int value = scan.nextInt();
stat.array[count] = value;
count++;
if(count > stat.array.length-1) {
break;
}
}

double avg = stat.average();
double max = stat.maximum();

System.out.println("Average: " + avg);
System.out.println("Maximum: " + max);
}
}

3.Write an algorithm to convert binary number to decimal number, then write a java method to implement this algorithm.

Answer:

Algorithm:
1.Input the binary number.
2.Multiply each digit of the binary number with the power of 2 and add each multiplication result.
3.The power starts from 0 to n-1 where n is the total number of digits in the binary number.

// Java program to convert binary to decimal

import java.io.*;

class Main {

// Function to convert binary to decimal
static int binaryToDecimal(String n)
{
String num = n;
int dec_value = 0;

// Initializing base value to 1 which is (2^0)
int base = 1;

int len = num.length();
for (int i = len – 1; i >= 0; i–) {
if (num.charAt(i) == ‘1’)
dec_value += base;
base = base * 2;
}

return dec_value;
}

public static void main(String[] args)
{
String num = new String(“1111”);
System.out.println(binaryToDecimal(num));
}
}

6.Write a method that find the factorial of a positive integer.



public static int fact(int n) {
if(n == 0) return 1;
if(n == 1) return 1;
return n * fact(n-1);
}

7.The Fibonacci series is given by: 1, 1, 2, 3, 5, 8, 13, etc, write a method that will compute the nth Fibonacci number

Answer:

public static int fib(int n) {
if (n==1) return 1;
if (n==2) return 1;
return fib(n-2) + fib(n-1);
}

8.Write a method that compute the power(x, n) where power(x, n)=xn

Answer:

public static int pow(int x, int n) {
if(n==0) return 1;
if (n==1) return x;
return x * pow(x, n-1);
}
