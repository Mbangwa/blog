1.Write the class Box that has instance variables length, width, and height and for each variable define accessors and mutators. The class also has method for computing volume of a box and surface area (assume the box is closed).

Answer:


// written by mbagwa.

public class Box {

private int length;
private int width;
private int height;

//accessor and mutators

public void setLength(int length) {
this.length = length;
}

public void setHeight(int height) {
this.height = height;
}

public void setWidth(int width) {
this.width = width;
}

public int getLength() {
return length;
}

public int getWidth() {
return width;
}

public int getHeight() {
return height;
}

// constructor

public Box(int h, int w, int l) {
setHeight(h);
setWidth(w);
setLength(l);
}

//volume

public int volume() {
return length * width * height;
}

// surface area (SurfaceArea=2lw+2lh+2wh)

public int surfaceArea() {
return (2*length * width) + (2*length*height) +
(2*width*height);
}
}
